## 1. Проблема

### Какая боль?

**Ежедневная неопределённость: "Это ещё можно есть?"**

Каждая семья в Америке стоит перед одним и тем же вопросом 1–3 раза в день. Ответ неизвестен, а ставки высоки: с одной стороны — пищевое отравление (48M случаев/год, CDC), с другой — выброшенная еда ($2,913/год на семью из 4, EPA 2025).

Проблема не абстрактная. Она конкретна:

| Ситуация | Частота | Результат |
|---------|---------|----------|
| "Курица лежит 3 дня — можно детям?" | 2–3 раза/неделю | Выбрасывает $8 курицы или рискует здоровьем |
| "Best by дата прошла — молоко испортилось?" | 1–2 раза/неделю | Выливает $4–6 молока, которое ещё нормальное |
| "Meal prep от воскресенья — сейчас пятница" | 1 раз/неделю | Выбрасывает 2 контейнера ($6–10) |
| "Клубника мягкая — это уже плесень?" | 1 раз/неделю | Выбрасывает всю упаковку ($5–7) |
| "Остатки ужина среды — можно разогреть?" | 2–3 раза/неделю | Выбрасывает при сомнении |

### Насколько боль острая?

**Frequency × Severity = Very High**

| Параметр | Значение | Источник |
|---------|---------|----------|
| **Частота** | Ежедневно, 1–3 раза/день | Reddit-анализ (MARKET-RESEARCH.md) |
| **Финансовый ущерб** | $2,913/год на семью ($56/неделю) | EPA 2025 |
| **Здоровье** | 48M случаев foodborne illness/год в US | CDC |
| **Эмоциональная нагрузка** | Тревога за детей (родители), вина за waste (миллениалы) | USER-PERSONAS.md |
| **Масштаб** | 84% американцев выбрасывают еду преждевременно по датам | NRDC |
| **Ухудшение** | Стоимость потерь по EPA 2025 почти удвоилась vs предыдущие оценки | EPA |
| **Рост отзывов продуктов** | +81% YoY (592 → 1,071 в 2024) | FDA |

### Как люди решают проблему сейчас?

| Текущее решение | Кто использует | Как работает |
|----------------|---------------|-------------|
| **Google-поиск** | Все | "Is chicken good after 3 days" → 50–110K запросов/мес |
| **Sniff test** | Все | Понюхать, посмотреть, иногда попробовать кончиком языка |
| **Reddit r/foodsafety** | Миллениалы/Gen Z | Фото еды + "is this still good?" → ждать ответ часы |
| **Даты на упаковке** | Пожилые, родители | Слепое следование "sell by" / "best by" как абсолюту |
| **Позвонить маме** | Молодые родители | Спросить совет по телефону |
| **"When in doubt, throw it out"** | Осторожные (родители, пожилые) | Правило по умолчанию — выбрасывать при любом сомнении |
| **Трекеры (Fridgely, NoWaste)** | Tech-savvy | Ручной ввод продуктов + уведомления о сроках |

### Почему текущие решения не работают?

| Решение | Почему не работает |
|---------|-------------------|
| **Google** | Противоречивые ответы (один сайт: 3 дня, другой: 5 дней). Требует 30+ сек. Не в моменте — нужно отложить продукт, достать телефон, набрать запрос |
| **Sniff test** | Субъективный. Salmonella, E. coli, Listeria **невидимы и не имеют запаха** (DOMAIN-RESEARCH.md). Ложное чувство безопасности |
| **Reddit** | Ответы через часы, не в момент принятия решения. Случайные люди, не эксперты |
| **Даты на упаковке** | 88% потребителей интерпретируют неправильно. "Sell by" — для магазина, не для потребителя. ~20% домашних отходов — из-за путаницы с датами (NPR/USDA 2025) |
| **Позвонить маме** | Мама не food scientist. Не доступна в 11pm. Даёт устаревшие советы |
| **"Throw it out"** | Работает для безопасности, но стоит $2,913/год. Не решает проблему — избегает её |
| **Трекеры** | Ручной ввод → 80% пользователей бросают через неделю (NoWaste отзывы). Не оценивают визуальное состояние |

## 2. Наше решение

### Ключевое обещание

**FreshCheck мгновенно отвечает на вопрос "Это ещё можно есть?" — по фото с камеры, используя AI + данные USDA, и предлагает что приготовить из того, что вот-вот испортится.**

### Value Proposition

> **Для** занятых родителей и food-conscious потребителей,
> **у которых** нет уверенности в свежести продуктов и которые теряют $2,913/год на выброшенной еде,
> **наш** AI-powered мобильный помощник
> **позволяет** мгновенно оценить свежесть по фото, отслеживать сроки и получать рецепты из того, что скоро испортится,
> **в отличие от** Google (медленно, противоречиво), sniff test (субъективно), трекеров (ручной ввод),
> **потому что** мы единственные, кто объединяет AI-фото оценку + трекинг сроков + рецепты в одном приложении, основанном на USDA-данных.

### 3 главных дифференциатора vs конкуренты

| # | Дифференциатор | Мы | Конкуренты | Источник |
|---|---------------|----|-----------:|----------|
| 1 | **AI фото-оценка свежести** | Фото → SAFE/CAUTION/DANGER за 3 сек для 15–20 категорий | USDA FoodKeeper: нет камеры. Fridgely: нет AI. RipenessFoody: только спелость | COMPETITORS.md — Gap Analysis |
| 2 | **Stateful + Proactive** | "My Fridge" знает ваш холодильник. Push-уведомление: "Курица — сегодня последний день" | Google/ChatGPT: stateless, нужно спрашивать. Трекеры: нет AI-оценки | RESEARCH-BRIEF.md — Risk 4 |
| 3 | **Полный food lifecycle** | Scan → Track → Cook → Save. Одно приложение для всего цикла | SuperCook: только рецепты. NoWaste: только трекинг. RipeOrNot: только авокадо | COMPETITORS.md — Positioning Map |

### "Aha-moment"

**Первый скан продукта, в котором пользователь сомневался.**

Пользователь фотографирует курицу, которую собирался выбросить. FreshCheck показывает **SAFE** (зелёный) с объяснением: "Визуальных признаков порчи не обнаружено. Куплена 2 дня назад — в пределах нормы USDA (3–4 дня для сырой курицы в холодильнике)."

Реакция: **"Я чуть не выбросил $8!"**

| Этап | Время | Что происходит |
|------|-------|---------------|
| Скачивание | 0 сек | TikTok/Facebook → App Store |
| Onboarding | 30 сек | "Сфотографируй первый продукт" (без регистрации) |
| **Aha-moment** | 45 сек | Первый скан → результат → "Вау, я бы это выбросил!" |
| Привычка | День 3–5 | Добавил 5+ продуктов в My Fridge, получил первый push |
| Конверсия | День 7–14 | Упёрся в лимит 5 сканов → Plus ($4.99) |

## 3. Validation Checklist

### Проблема подтверждена данными

- [x] $2,913/год потерь на семью (EPA 2025) — финансовая боль
- [x] 48M случаев foodborne illness/год (CDC) — health-боль
- [x] 84% американцев выбрасывают еду преждевременно (NRDC) — масштаб
- [x] 88% неправильно интерпретируют даты (NPR/USDA 2025) — корневая причина
- [x] r/foodsafety: ежедневные фото с "is this good?" — наблюдаемое поведение
- [x] Поисковый спрос: "how long does chicken last" — 50–110K/мес (MARKET-RESEARCH.md)
- [x] Food waste запросы удвоились за 5 лет (Google Sustainability)

**Вердикт: ПОДТВЕРЖДЕНА** — проблема универсальна, ежедневна, измерима в деньгах и здоровье.

### Аудитория достаточно большая

- [x] TAM: $4.2B (131M домохозяйств US × потенциальная готовность платить $32/год)
- [x] SAM: $840M (~20% smartphone-savvy, food-conscious домохозяйств)
- [x] SOM Year 3: $12–18M (200–300K подписчиков × $55/год)
- [x] 32M семей с детьми до 18 лет — primary persona
- [x] Yuka доказала: food scanning app может достичь 80M пользователей
- [x] AI-driven food segment: CAGR 28–30.9% (MARKET-RESEARCH.md)

**Вердикт: ДОСТАТОЧНО** — рынок большой и растущий. Даже 0.15% SAM = $12M+ ARR.

### Конкурентов можно обойти

- [x] Market Saturation Score: 3/10 (COMPETITORS.md)
- [x] Лучший конкурент — USDA FoodKeeper с рейтингом 3.2 ★
- [x] CozZo закрывается (декабрь 2025) — вакуум
- [x] Ни одно приложение не комбинирует AI-фото + трекинг + рецепты (Gap Analysis)
- [x] Правый верхний квадрант positioning map (AI + широкий scope) — пуст
- [x] Фрагментированный рынок мелких инди-приложений с 3–4 ★
- [x] Yuka валидирует модель, но не адресует freshness use case

**Вердикт: МОЖНО** — конкуренция практически отсутствует в комбинированной категории.

### Техническая реализация возможна

- [x] VGG16: 97% точность на фруктах/овощах (Nature Scientific Reports, 2025)
- [x] VGG19: 98.5% точность на мясе (Published research, 2024)
- [x] MobileNetV2: 94%+ точность при on-device inference (DOMAIN-RESEARCH.md)
- [x] Inference <500ms на mid-range phones — подтверждено
- [x] USDA FoodKeeper database — публично доступна (400+ продуктов)
- [x] Expo SDK 55 + React Native + CoreML/TFLite — production-ready стек
- [x] Adapty для подписок — проверенное решение
- [x] OpenAI API для рецептов — доступно

**Вердикт: ВОЗМОЖНО** — технология готова, accuracy достаточна для MVP с 15–20 категориями.

## 4. Risks to Problem-Solution Fit

| Риск | Severity | Вероятность | Митигация |
|------|----------|-------------|-----------|
| AI ошибается → пользователь болеет | Критическая | Средняя | Никогда "safe" абсолютно. "Appears fresh" + disclaimer. False negatives >> false positives |
| Lab accuracy ≠ real-world accuracy | Высокая | Высокая | 15–20 категорий на 95%+, не 50 на 80%. Confidence score. Default to CAUTION |
| "Try once and forget" | Высокая | Средне-высокая | My Fridge + push — core retention. Weekly Fridge Report. Recipes = daily habit |
| Google/Apple встроят аналог в ОС | Высокая | Средняя (1–3 года) | Moat = stateful (знает холодильник) + proactive (push). Google Lens = stateless |
| Пользователи over-rely на AI | Средне-высокая | Высокая | Постоянный disclaimer. Raw poultry → всегда "cook to 165°F". Education в приложении |
