## Трудности

- **Эмулятор падал 8+ раз** во время тестирования. Каждый раз: kill → remove locks → restart → boot wait → reverse tunnel → relaunch app. В среднем 3-5 минут потери на каждый краш.
- **pm clear** (для теста онбординга) сбрасывает все данные — при следующих перезапусках эмулятора приложение каждый раз начинало онбординг заново.
- **Expo Dev Menu** мешал: кнопка X на Paywall перехватывается экраном разработчика (B08). Решение: использовать "Go home" из dev menu.
- **BACK key** в Q6 скрывает клавиатуру И закрывает экран — нужно использовать свайп вместо BACK для скрытия клавиатуры.
- **Координаты нажатий**: uiautomator dump нестабильно работал (иногда возвращал пустой XML). Решение: двойной слеш в путях (`//sdcard//ui.xml`).
- **API ограничения**: скриншоты 1080x2400 (~180-200KB) иногда отклонялись API в режиме многих открытых изображений.
