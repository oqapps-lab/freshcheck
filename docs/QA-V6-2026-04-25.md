## Bugs found + fixed

### #1 — Fridge filter pills hardcoded; Poultry/Bakery/Pantry unfilterable [low]

**Symptom:** Filter row shows only "All / Produce / Dairy". Mock fixtures
include Chicken Breast (`category: poultry`), and the schema enum
allows `bakery` + `pantry` too. Selecting any non-All filter would
silently hide poultry items.

**Fix:** Derive pill set from categories actually present in `items`.
Result: when fridge has dairy + poultry + produce items, pills are
"All / Produce / Dairy / Poultry". Adding a bakery item would add
a Bakery pill, etc. — pills are always relevant to the user's data.

**File:** `app/(tabs)/fridge.tsx:21-49`. tsc clean.
