## 1. Прямые конкуренты (AI-анализ свежести)

### 1.1 USDA FoodKeeper

| Поле | Данные |
|------|--------|
| **Платформа** | iOS + Android + Web |
| **Цена** | Бесплатно |
| **Рейтинг** | 3.2 ★ (Google Play) |
| **Скачивания** | ~182,000 (с 2015) |

## 4. Бенчмарки — смежные лидеры

### 4.1 Yuka (ключевой бенчмарк)

| Поле | Данные |
|------|--------|
| **Платформа** | iOS + Android |
| **Цена** | $10–50/год (pay what you can) |
| **Рейтинг** | 4.8+ ★ |
| **Пользователи** | 80+ млн |
| **Скачивания** | ~1M iOS + 600K Android/мес |
| **Выручка** | $7.3M (2024), 98% от подписок |
| **Ключевые фичи** | Штрихкод → health-рейтинг, алерты на добавки, диетические фильтры, офлайн |
| **Слабости** | Только штрихкоды (нет фото AI), только здоровье (не свежесть), не определяет порчу |

**Значение для FreshCheck:** Yuka доказала, что consumer food scanning app может достичь $7.3M+ ARR на подписках без рекламы. FreshCheck адресует другой, но равно убедительный use case.

## 5. Positioning Map

```
                    ВЫСОКАЯ СЛОЖНОСТЬ (AI)
                           |
     Fridge Leftovers AI   |   ★ FreshCheck ★
     RipenessFoody         |   (единственный в этом квадранте)
     RipeOrNot AI          |
                           |
УЗКИЙ SCOPE ───────────────┼─────────── ШИРОКИЙ SCOPE
                           |
     BEEP, Kiff, Best      |   Yuka (health scanning)
     Before                |   SuperCook (рецепты)
     USDA FoodKeeper       |   Fridgely (трекинг)
                           |
                    НИЗКАЯ СЛОЖНОСТЬ (Manual)
```

**Ось X:** Scope (количество функций — от 1 фичи до полного набора)
**Ось Y:** Технологическая сложность (ручной ввод vs AI/камера)

**Ключевой вывод:** Правый верхний квадрант (AI + широкий scope) — **полностью пуст**. FreshCheck будет первым.

## 7. Market Saturation Score

### Оценка: **3/10** (низкая насыщенность)

**Обоснование:**

| Фактор | Оценка | Комментарий |
|--------|--------|-------------|
| Количество прямых конкурентов | 2/10 | 3-4 мелких приложения с AI-фото, все нишевые |
| Качество конкурентов | 2/10 | Лучший — FoodKeeper с 3.2 ★. Нет сильного продукта |
| Присутствие крупных игроков | 3/10 | Yuka — в смежном рынке. Google/Apple пока не вошли |
| Рыночная консолидация | 2/10 | Фрагментированный рынок мелких инди-разработчиков |
| Барьеры входа | 4/10 | AI-модели требуют данных, но технология доступна |
| Лояльность аудитории | 3/10 | У существующих приложений нет сильной лояльности |

**Итог:** Рынок крайне фрагментирован с низкокачественными игроками. Нет доминанта. CozZo закрывается, создавая дополнительный вакуум. **Идеальные условия для входа.**
